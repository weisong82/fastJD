#解决写JD（招聘需求）的头疼问题：
1.  通过从BAT招聘网站爬取的几万条JD，以及其他xx站的百万条JD 作为源
2.  使用了结巴jieba分词，对文本进行处理，变成语料。
3，  使用了gemsim 框架，做了词袋模型，对查询的词组和库里面的做相似度匹配。

#数据源：
